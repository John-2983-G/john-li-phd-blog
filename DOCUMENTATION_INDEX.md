## 🆘 问题排查流程

```
遇到问题
   ↓
检查浏览器是否显示正常？
   ├─ 不正常 → 查看 README_PERSONALIZATION.md 的故障排除
   └─ 正常 → 问题可能在配置上
   ↓
确认保存和刷新了吗？
   ├─ 没有 → Ctrl+S 然后 Ctrl+R
   └─ 已做 → 检查代码是否有语法错误
   ↓
代码是否有错？
   ├─ 不确定 → 打开 F12 查看控制台
   └─ 有错 → 参考 CUSTOMIZATION_GUIDE.md 的示例代码
   ↓
还是不行？
   └─ 查看 README_PERSONALIZATION.md 的常见问题
```
